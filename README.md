# now-ng28-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，旨在提供简单、可访问的信息展示。仓库不针对任何特定域名或网站，仅作为页面存档与分发使用。

## 目录结构

```
.
├── index.html          # 页面入口
├── pages/              # 独立页面目录
│   ├── page1.html
│   └── page2.html
└── assets/             # 公共资源目录
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库主页，可提供页面索引或导航。
- `pages/`：存放所有独立 HTML 页面，每个页面可独立访问。
- `assets/`：存放页面所需的样式、脚本、图片等公共资源。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开。
- 页面内容不包含任何形式的推广、营销或诱导性信息。
- 归档页面仅供信息展示与参考，不保证内容的时效性或准确性。
- 页面资源尽量内嵌或使用相对路径，避免依赖外部链接。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来补充或修正页面内容。
- 新增页面请放入 `pages/` 目录，并保持文件命名清晰。
- 请勿在页面中插入任何形式的广告或跟踪代码。
- 本仓库由社区维护，不设专职管理者。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中打开 `index.html` 或 `pages/` 下的任意页面。
3. 也可通过 GitHub Pages 等服务部署后直接访问。
